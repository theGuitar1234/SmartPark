# Admin Workflow Guide

## Standard setup workflow

1. Sign in as admin.
2. Open Forms Library.
3. Create a parking lot.
4. Create zones under the lot.
5. Create individual slots or use Bulk slot creation.
6. Register sensors and assign them to slots.
7. Create tariff plans for default, EV, VIP, or zone-specific pricing.
8. Use Dashboard and Zone Management for live operations.

## Daily operation workflow

1. Check Dashboard KPI cards and alerts.
2. Review sensor health and unpaid exposure.
3. Use the occupancy map to inspect problematic slots.
4. Start sessions through optimized assignment.
5. Simulate sensor updates if presenting without hardware.
6. Exit sessions when vehicles leave.
7. Process payments and show receipts.
8. Review Reports for utilization and revenue.

## Incident workflow

If a vehicle must be released quickly, use Dashboard > Emergency exit. This exits an active session, calculates the fee, releases the slot, and records activity.

If a slot is physically unavailable, open the slot drawer and mark Maintenance or Disabled. The optimizer excludes that slot immediately.
