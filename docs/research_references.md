# Research References Used

The implementation was aligned with official documentation for these areas:

- FastAPI application structure, path operations, and APIRouter usage.
- Pydantic v2 model validation and field validation.
- Flet controls, dialogs, navigation, and web app execution concepts.
- SQLite foreign key enforcement and constraints.
- Mermaid ER diagram syntax.
- DBML relationship syntax.

Official docs should be preferred over blogs when upgrading the project because Flet and FastAPI APIs can change.
