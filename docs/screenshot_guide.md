# Screenshot Guide

Run the backend and frontend first, then open the Flet web app at `http://127.0.0.1:8550`.

Capture these screens for presentation slides or report evidence:

1. Login screen with branded SmartPark panel and demo credentials.
2. Dashboard showing KPI cards, quick actions, occupancy map, revenue visual, active sessions, and alerts.
3. Zone Management with lot/zone selectors, zone cards, and the slot grid.
4. Slot Detail Drawer after clicking a slot in the occupancy map.
5. Sessions & Payments with filters, summary cards, sessions table, and row actions.
6. New Session modal with optimizer preview.
7. Exit Session modal and the calculated fee feedback.
8. Payment Processing modal with receipt preview.
9. Receipt View modal after successful payment.
10. Forms Library showing the six management form cards.
11. System Configuration showing health cards, sensor network, tariff plans, and bulk create action.
12. Bulk Import Infrastructure modal after previewing generated slots.
13. Reports screen with summary cards, zone utilization, and payment history.
14. FastAPI `/docs` page showing SmartPark domain endpoints, not generic items CRUD.

Recommended capture size: 1440 x 900 browser window. Use a clean seeded database so the dashboard looks populated.
