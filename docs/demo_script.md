# SmartPark Demo Script

1. Start the backend and frontend.
2. Login with `admin@smartpark.com` / `admin1234`.
3. On Dashboard, point out revenue, occupancy, active sessions, available slots, sensors online, and payment due.
4. Click a slot tile. Explain that the side panel is operational: it shows status, scoring, current session, sensor state, conflict warnings, and history.
5. Simulate a sensor update from the slot panel and return to Dashboard to show alerts and state changes.
6. Open Zone Management. Use lot/zone selectors, search, and status filter. Click a zone card and then a slot.
7. Add a zone or slot. Show validation and success feedback.
8. Use Bulk Create. Preview a range, show duplicate detection if needed, then confirm a clean import.
9. Open Sessions & Payments. Start a new session using the optimizer preview.
10. Exit the session. Show fee calculation from the tariff snapshot.
11. Process payment. Open the receipt view.
12. Open System Configuration. Edit a sensor or tariff, simulate a sensor, and explain that fake decorative switches were removed.
13. Open Reports. Generate summary, payment, and session CSV exports.
14. End by opening FastAPI docs and showing SmartPark-specific endpoints.
